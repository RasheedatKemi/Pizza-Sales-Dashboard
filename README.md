# Pizza Sales Analysis

This repository contains an analysis of a pizza sales dataset to provide insights into business performance and customer trends. The dataset was cleaned and analyzed in **Excel**, and a dashboard was created to visualize key metrics.

## Dataset Overview

The dataset includes:
- Order date
- Pizza category and size
- Units sold per order
- Order value 

## Key Analysis Insights

- Average orders per day and total orders
- Total units sold
- Percentage of sales by pizza category
- Sales distribution by pizza size
- Daily orders trend
- Top 5 best-selling pizzas and worst-selling pizzas
- Insights sliced by order date

## Tools Used

- **Excel** for data cleaning, analysis, and dashboard creation
